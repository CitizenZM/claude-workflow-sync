All outreach and project workflows must follow this naming convention:

**Pattern**: `[Platform] [Brand] [Region] Affiliate Outreach [MonthDDYYYY]`

**Examples**:
- Awin Rockbros US Affiliate Outreach April152026
- Impact Levoit US Affiliate Outreach May012026
- CJ Cosori CA Affiliate Outreach June102026
- Awin TCL EU Affiliate Outreach July202026

**Why:** User wants consistent, scannable naming across all outreach projects for quick identification in Obsidian and skill files.

**How to apply:** When creating any new outreach workflow, skill, command, report, or ledger file, use this naming pattern. Apply to skill names, command titles, report headers, and Obsidian note titles.
