MyGreenhouse (my.greenhouse.io) will rate-limit and stop rendering the search input if queries are fired too rapidly. After ~13 rapid searches the site blocks further interaction.

**Why:** The site detects bot-like behavior from fast sequential searches with no natural pauses.

**How to apply:** Always use human-like delays (5-12s random) between searches, re-navigate to /jobs page fresh each time, type characters one by one with random keystroke delays (50-150ms), and save progress incrementally so interrupted runs can resume. Never use Google to search Greenhouse — stay within the my.greenhouse.io site.
